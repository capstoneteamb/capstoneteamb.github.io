Capstone Management Webapp - Overview

A web app designed to help with the PSU Capstone classes. At minimum, it will help with the 360 review process. At best, it will help with Capstone team lead and team management as well. The web app is to be hosted on a a stock Linux installation using Apache2 or Nginx, will use Postgres as the database, and will be written in Python. Security is a high priority, and must be built in from the ground up.
